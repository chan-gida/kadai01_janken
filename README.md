とある串カツ屋のチンチロ

## ②課題内容（どんな作品か）

- ハイボールの無料券を目指し、サイコロを2種類転がすアプリ
  - １のゾロ目：メガハイボール 無料
  - ゾロ目　　：ハイボール １杯無料
  - 偶数　　　：ハイボール　半額
  - 奇数　　　：倍額払ってメガハイボール
  ※ CSSで貼り付けた画像と条件が一致してませんでした
- 実は「倍額払ってメガハイボール」が出る確率が7割の悪徳商売アプリ

- サイコロの2種類の合計値が出る確率の調整
